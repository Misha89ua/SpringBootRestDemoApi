Даний проект представляє собою Spring Boot REST сервіс із викоританням
Spring Data, JPA, Maven, BD in-memory H2. 
Щоб запустити даний проект достатньо імпортувати його через IDE Intellig Idea, 
або завантажити архів. Нижче буде силка на Postman-колекцію запитів для перевірки 
роботи проекту і описання запитів.



https://www.getpostman.com/collections/5d0818f0be1d654a6150  -  лінк на колекцію запитів для Postman


Усі запити мають свій опис.
Кольори представлені як Enum, їх потрібно вказувати великими літерами одним із наведених нище варіантів:
BLACK, BLUE, RED, GREEN.
